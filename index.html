🔥 Firebase Connected — Where Your Data Lives

All data saves to Firebase Firestore. To view it: go to console.firebase.google.com → your project → Firestore Database. You will see these collections:

📂 Firestore Data Collections

CollectionWhat's stored thereHow to view
participantsEvery registered user — name, email, points, tasks done, streakFirestore → participants → click any document
participants/{uid}/completionsEach task, quiz, poll completed by a participant (sub-collection)Firestore → participants → click a user → completions
pollResponsesEach participant's poll answer + mindfulness category (positive/neutral/needs-improvement) stored in completions sub-collectionFirestore → participants → {uid} → completions → poll_{taskNum}
batchesYour program batches — name, active week, active taskFirestore → batches
tasksAll 40 task details — title, message, image URL, points, statusFirestore → tasks
themesWeekly theme titles and messages (weeks 1–8)Firestore → themes
quizzesQuiz questions and correct answers per task numberFirestore → quizzes
pollsPoll questions and point values per task numberFirestore → polls
surveyQuestionsCustom survey questions you add via Survey BuilderFirestore → surveyQuestions
adminsAdmin user IDs (only you)Firestore → admins
configPoints settings (task=10, quiz=5, poll=3, streak=2)Firestore → config → global

💡 Tip: In Firestore, you can click any collection name to browse all documents. Click any document to see all fields. Use the Export CSV buttons in Participants and Survey Builder panels to download spreadsheets.

Admin Help

1

Admin Sign In

Use your admin email and password — same as any participant login but your account must also have its UID listed in the Firestore "admins" collection. See the SOP for setup instructions.

2

Create a Batch

Batches → fill name, start date, institution → Create Batch. Participants select this batch when registering.

3

Add Task Content

Tasks → enter task number → fill Title, Description, Resource text. Quiz Questions and Poll Questions panels use the same task number to link content.

4

Advance Tasks

Batches → click your batch → click "Advance Task" to unlock the next task for all participants in that batch.

5

View All Data

Overview panel shows a full table of all Firestore collections. For raw data: console.firebase.google.com → your project → Firestore Database.

6

Poll Categories

When saving a poll, assign a category (Positive / Neutral / Needs Improvement) to each option. This drives the Mindfulness Insights section on participant dashboards.
